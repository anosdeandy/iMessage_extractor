  • Download the file above
  • Open Terminal (Applications → Utilities → Terminal)
  • Run: python3 ~/Downloads/imessage_extract.py
  • Find your output files in ~/Desktop/imessages

SMS + iMessage merged: The script automatically finds all handles for each number (both SMS and iMessage) and merges them into one chronological file.

Permission error? Go to System Settings → Privacy & Security → Full Disk Access, add Terminal, then quit and reopen it before running again.

0 messages returned? Try the number without the +1 prefix, or use the contact's Apple ID email address instead.

Missing old messages? In Messages app go to Messages → Settings → iMessage, enable "Messages in iCloud", scroll back in the conversation to trigger sync, then re-run.

Still seeing blanks? The script decodes Apple's newer binary message format automatically. Any remaining blanks are tapbacks or system messages with no text content.
